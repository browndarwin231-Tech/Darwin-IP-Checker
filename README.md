# Darwin IP Checker

A beginner Python networking project that identifies common IP addresses and introduces basic networking concepts.

## Features

- Detects Localhost (127.0.0.1)
- Detects a common Router IP (192.168.1.1)
- Identifies external IP addresses
- Beginner-friendly Python code

## Example

Input:

```
127.0.0.1
```

Output:

```
Localhost
Created by Darwin Brown
```

## Skills Used

- Python
- User Input
- Conditional Statements (if/elif/else)
- Basic Networking Concepts
